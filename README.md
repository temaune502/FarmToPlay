# FarmToPlay

`FarmToPlay` is a Kotlin Android game starter built with Jetpack Compose.
This repository now contains:

- a working project scaffold for Android
- menu screen placeholders
- a simple prototype gameplay loop
- an architecture draft for team development

## Project goals

The current build is a foundation for a casual mobile game in a farm theme.
The prototype includes:

- Main menu
- Mode selection
- Settings screen
- Lightweight tap-based gameplay placeholder

## Suggested stack

- Kotlin
- Android SDK
- Jetpack Compose
- Navigation Compose
- Material 3
- AndroidX Lifecycle ViewModel

## Project structure

- `app/` Android application module
- `docs/ARCHITECTURE.md` team-friendly architecture plan

## Next recommended steps

1. Open the project in Android Studio.
2. Let Android Studio generate or sync the Gradle wrapper if needed.
3. Install Android SDK 35 and required build tools.
4. Replace placeholder gameplay with your real core loop.
5. Split features into separate Gradle modules when the team grows.
